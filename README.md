# 🚀 TradeHorizon – Decentralized Exchange (DEX)

TradeHorizon is a modern **Decentralized Finance (DeFi)** platform built on the **Solana Blockchain** that enables users to securely trade cryptocurrencies without relying on centralized intermediaries.

The platform focuses on:
- 🔐 Security
- ⚡ Fast Transactions
- 🌐 Transparency
- 💸 Low Transaction Fees
- 👛 User Asset Ownership

# 🏗️ Project Overview

Traditional crypto exchanges operate using centralized infrastructure where user funds and transactions are controlled by third parties.

TradeHorizon solves this issue using blockchain technology, allowing users to:
- Maintain ownership of digital assets
- Execute secure peer-to-peer transactions
- Perform decentralized token swaps
- Access transparent blockchain settlement

The platform uses the **Solana Blockchain** because of:
- High throughput
- Low gas fees
- Fast transaction confirmation
- Scalability for dApps

# 🔄 Working Mechanism

TradeHorizon uses an **Automated Market Maker (AMM)** model for token swapping.

## Token Swap Flow

```bash
1. User connects wallet
2. Selects token pair
3. System fetches liquidity routes
4. Best route calculated using Jupiter Aggregator
5. User signs transaction
6. Transaction submitted to Solana blockchain
7. Blockchain confirms transaction
8. Updated portfolio balance displayed
```
